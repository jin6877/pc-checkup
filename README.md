# 🩺 PC 점검 (PC Checkup)

지금 켜둔 프로그램을 점검해서 **내 PC에 맞는 램·CPU 사양을 추천**해주는 Windows 데스크톱 진단 도구.

> "프로그램을 잔뜩 펼쳐놓고 컴퓨터를 잘 안 끄는데 렉이 심하다" — 이럴 때 램을 늘려야 하는지 답을 줍니다.

## 무엇을 하나요?

- 현재 **실행 중인 모든 프로그램**의 메모리·CPU 사용량을 읽어 대표 앱 단위로 묶어 정리
- 램 사용률·스왑(디스크 대체) 사용량을 보고 **메모리 압박 정도**를 판정
- 사용 패턴에 여유를 두고 계산해 **알맞은 램 용량(8/16/32/64GB)** 추천
- 메모리를 많이 먹는 앱 TOP 리스트 제공

모든 분석은 **로컬(내 PC)에서만** 이뤄지며 어떤 데이터도 외부로 전송하지 않습니다.
점검 전, 무엇에 접근하는지 사용자에게 안내하고 동의를 받습니다.

## 개발/실행

```bash
npm install
npm start          # Electron 앱 실행
npm run build:win  # Windows 설치파일(.exe) 빌드 → dist/
```

## 배포 구조

- **앱**: Electron. `v*` 태그를 push하면 GitHub Actions(windows 러너)가 `.exe`를 빌드해 릴리스에 첨부.
- **다운로드 사이트**: [`web/`](web/) — 정적 랜딩 페이지(Vercel 배포). 릴리스의 최신 `.exe`로 연결.

## 폴더

```
lib/analyze.js   진단 로직(사양 추천 엔진)
main.js          Electron 메인 · systeminformation로 시스템 수집
preload.js       IPC 브릿지
renderer/        앱 화면(동의 → 점검 → 결과)
web/             다운로드 랜딩 페이지
```

## 설치 시 보안 경고에 대해

코드 사이닝 인증서가 없어 Windows SmartScreen 경고가 뜰 수 있습니다(개인 무료 앱이라 정상).
`추가 정보 → 실행`으로 설치할 수 있습니다.
